

Project

A Dashboard built with React, JavaScript, and CSS.


Installation and Setup Instructions

Clone down this repository. You will need node and npm installed globally on your machine.

Installation:

npm install

To Start Server:

npm start

To Visit App:

localhost:3000


node version : v16.16.0

npm version : 8.11.0
